== Description ==

实现复制视频超链接解析播放bilibili视频、优酷视频、youtube视频、腾讯视频、西瓜视频，并且可以统一控制视频大小，操作非常简单不需要去复制视频分享代码，直接到游览器复制URL链接即可，添加到编辑器里，非常方便用户使用。

## 插件功能

* 支持复制bilibili视频URL链接，解析视频播放，操作非常简单，无需要复制分享代码（如不明白，可看截图）
* 支持bilibili解析格式一：https://www.bilibili.com/video/BV16K411Z7Kv/?带参数
* 支持bilibili解析格式二：https://m.bilibili.com/video/BV16K411Z7Kv?带参数
* 支持后台自定义视频宽度
* 支持后台自定义视频高度
* 支持复制优酷视频URL链接，解析优酷视频播放，操作非常简单，无需要复制分享代码+
* 支持优酷解析格式一：https://v.youku.com/v_show/id_XNTk5MjA0OTM5Ng==.html?带参数
* 支持复制youtube视频URL链接，解析youtube视频播放，操作非常简单，无需要复制分享代码
* 支持youtube视频解析格式一：https://www.youtube.com/watch?v=EN_HzPZVfNE
* 支持youtube视频解析格式二：https://youtu.be/HOI6Q1y9goU
* 支持复制腾讯视频URL链接，解析腾讯视频播放，操作非常简单，无需要复制分享代码
* 支持腾讯视频解析格式一：https://v.qq.com/x/page/r3559zieqqy.html
* 支持腾讯视频解析格式二：https://v.qq.com/x/cover/zowhuqn2hvz7x5l/k0797bid33m.html
* 支持复制西瓜视频URL链接，解析西瓜视频播放，操作非常简单，无需要复制分享代码
* 支持西瓜视频解析格式一：https://www.ixigua.com/7281577821901128227?带参数
* 支持禁止播放器跳转源视频里，比如禁止跳转到bilibili等
